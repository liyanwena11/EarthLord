## 💰 定价心理学 - 为什么这样定价？

### VIP定价的心理学

#### 原则1: 阶梯定价 (Price Tiering)

```
¥6    (试用)  ← 低价入场，降低决策难度
¥12   (月)    ← 标准价格，大多数人选择
¥28   (季)    ← 折扣促销，鼓励长期承诺
¥88   (年)    ← 最优价格，忠诚用户选择
```

**心理效应**: 用户会对比价格，看到¥28比¥36便宜25%，觉得"很划算"

#### 原则2: 相关对比 (Price Anchoring)

```
VIP月会员 ¥12 → 每天 ¥0.4 (咖啡价格)
VIP年会员 ¥88 → 每天 ¥0.24 (水价格)
```

**心理效应**: 用户对小数字不敏感，需要转换为"每天"成本

#### 原则3: 滑动定价 (Price Optimization)

```
首月试用 ¥6  (相当于50% 折扣) ← 降低试用成本
续费价格 ¥12 (标准价格)       ← 用户已上瘾，易接受
```

**心理效应**: 用户会先体验低价，再按常价续费

## ⚠️ 常见陷阱与避免

### 定价陷阱

| 陷阱 | 后果 | 避免方法 |
|------|------|--------|
| **定价过低** | 用户觉得便宜但不重视权益 | 确保价格与价值对等，不低估权益 |
| **定价过高** | 转化率下降，用户流失 | 进行A/B测试，参考市场数据 |
| **选择过多** | 用户决策困难，反而不买 | 限制为3-5个选项，突出推荐 |
| **权益不清** | 用户不明白买了什么 | 清单式展示，用图表对比 |

### 这不做的错误

| 错误 | 理由 | 正确做法 |
|------|------|--------|
| ❌ 产品取消后台无通知 | 违反App Store规则 | ✅ 提前3-7天发送续费通知 |
| ❌ 隐藏首次订阅的试用期 | 欺骗用户，导致投诉 | ✅ 清晰展示"首月¥6，后续¥12" |
| ❌ 没有取消按钮 | 会被App Store拒审 | ✅ 设置 → 订阅管理 → 取消选项 |
| ❌ 宣传"无限"或"永久" | 违反App Store广告规则 | ✅ 使用"30天"、"1个月"等具体时长 |
| ❌ 优先推荐高价产品 | 降低信任度 | ✅ 根据用户等级推荐对等产品 |
